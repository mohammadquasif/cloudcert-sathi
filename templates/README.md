# Templates — cloudcert-sathi

This folder contains blank templates for contributing to this repo.

---

## Available Templates

| Template | Use This For |
|---|---|
| `study-guide-template.md` | Writing chapter notes and study guides |
| `question-template.md` | Writing original practice questions |

---

## How to Use

1. Pick the right template for what you want to create
2. Make a **copy** of the template (don't edit the original)
3. Save your copy in the correct folder (e.g., `aws-cloud-practitioner/03-core-services/`)
4. Fill in all sections
5. Submit a Pull Request

See `contribution-guide.md` for detailed steps.
