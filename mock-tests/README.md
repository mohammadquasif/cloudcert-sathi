## Rules for Taking Mock Tests

1. **Set a timer** — 90 minutes for AWS CCP (65 questions)
2. **No notes, no Googling** during the test
3. **Attempt every question** — no negative marking
4. **Review all answers after** — especially the ones you got right by guessing

| Score | What It Means |
